# Native Editable PPTX Case Brief

Title: Native objects survive the handoff
Audience: Presentation-producing agents and maintainers evaluating PPTX editability
Moment: A benchmark case proving that a generated `.pptx` contains native objects rather than flattened slide images
Desired audience shift: Treat editability as an evidence-backed contract
Decision or learning outcome: Require object inspection, rendered previews, and a documented exception list before claiming a deck is editable
Main promise: The Knowledge Cat native lane can generate and verify editable text, shapes, charts, tables, and notes from a deck plan
Scope: Six-slide self-contained benchmark case with no external facts or image assets
Output lane: native-pptx
Format: 16:9 PowerPoint
Page count: 6
Source materials: Knowledge Cat native PPTX production contract and benchmark gates
Design posture: Editorial technical minimalism; warm paper, dark ink, coral proof accent
Must include: Native chart, native table, speaker notes, re-import inspection, full-slide previews, contact sheet, editability report
Must avoid: Full-slide screenshots, decorative stock imagery, unsupported market claims, placeholder text
QA level: Benchmark case-study contract
